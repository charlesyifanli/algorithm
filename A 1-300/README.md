## 0001 two sum
ideas:<br>
```
01----O(n²)
肯定要遍历数组。
指针到达某一个数组元素（下标和值）后，与目标值做差。
从当前元素的下一个开始，直到最后一个元素，与差值比较。
```
```
02----O(n)
肯定要遍历数组。
利用键值对，key（数组的值） value（数组下表）。
指针到达某一个数组元素（下标和值）后，与目标值做差。
判断差值是否在map里面，不在就把这个键值对放入map。
```
